Theme Installation Errors

During installation you can have some errors. Here the most popular errors you can get:

  • Error "Parse error: syntax error, unexpected ]"
    Solution: you need to change the PHP version on your hosting. Minimal version 5.4. You can change it in the PHP settings of your hosting.

  • The package could not be installed. The theme is missing the style.css stylesheet
    Solution: you've forgotten to unzip the main archive. It consist: theme, license, offline documentation and demo content.

  • «Are You Sure You Want to Do This?»
    Solution: you need to increase your PHP parameter upload_max_filesize to 256M. You can change it in the PHP settings of your hosting.

  • "Error Execution Failed"
    Solution: you need to increase the PHP parameter: max_execution_time to 400-600 sec. You can change in the PHP settings of your hosting.

Problems with the Theme License Activation

In some reasons you can have problems with the Theme activation:
1) "Purchase code is wrong" - check that you activate the main theme (not the child). After go Theme Options->License, insert you purchase code (check that it hasn't spaces after (you can mistake copy it with)), click "Save" and wait 5-7 sec. After that, click "Register the code". Some servers have delay and waiting 5-7 sec. solve this problem.
2) "Something wrong, please try later" - that's mean that you need to try to activate your license later because Envato license system is not active at the moment. It happens very seldom, but if after 1 hour you'll have the same notice, please contact our support team via support form from your ThemeForest account.

Initial Theme Setup

Ok, so far we have installed the theme, we have installed the plugins and imported the demo data, whats next?

The final step in getting your new theme setup and ready for you to edit as you wish is to setup your home page and menu. To begin, head to Settings > Reading, here you will need to set your Front Page.

By default WP is set to show your latest posts, however to make your theme just like our demo you should select a static front page from the pages provided in the demo data.
